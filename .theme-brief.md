## (c) Glow -> shadow/wash conversion rules
On light, depth = contact shadows + hairlines; color = flat tint washes. NEVER blurred neon glows.

1. Colored radial glow divs (blur-3xl pools behind objects), neon hex at alpha A -> same geometry, NO blend mode, new deep hex at roughly A*0.45:
   - rgba(56,232,255,A) -> rgba(0,109,124,A*0.45)   [cyan]
   - rgba(255,46,154,A) -> rgba(184,13,98,A*0.45)   [magenta]
   - rgba(255,122,24,A) -> rgba(168,67,0,A*0.5)     [tangerine]
   - rgba(67,102,255,A) -> rgba(44,69,200,A*0.45)   [cobalt]
   - rgba(214,255,59,A) / rgba(182,255,46,A) [acid/lime pools]: if it is a SIGNATURE pool (Hero blob, iPhone ambient) -> marker wash rgba(214,255,59,0.18–0.22); if it is subtle atmosphere -> rgba(93,107,0,0.06)
2. box-shadow glows on elements (`0 0 Npx -Mpx <hue>`) -> flat halo `0 0 0 4px <newHue at 15% alpha>` (e.g. #b80d6226). No blur.
3. Object elevation: use the `.lift` class (0 1px 2px rgba(28,27,23,0.05), 0 16px 40px -24px rgba(28,27,23,0.25)).
4. Hero objects/devices: contact shadow `0 30px 60px -28px rgba(28,27,23,0.38)`.
5. Tiny dot glows (`0 0 8px var(--color-acid)`) -> `0 0 0 4px rgba(93,107,0,0.18)`.
6. Shadows are for OBJECTS only (devices, cards, glass). Sections separate via tint bands + hairlines, never shadow walls.
